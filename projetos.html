Manipulação e Análise de Arrays com NumPy

Script em Python focado na criação e exploração de estruturas de dados multidimensionais (matrizes) utilizando a biblioteca NumPy. O código demonstra a instanciação de arrays através de diversos métodos nativos (como arange, zeros, ones, linspace e empty) e a alteração de suas dimensões utilizando a função reshape. Além disso, o projeto inspeciona os atributos fundamentais dessas estruturas — como formato (shape), número de dimensões (ndim), tipagem de dados (dtype) e alocação de memória —, evidenciando o domínio de ferramentas essenciais para a área de Ciência de Dados e computação científica.
