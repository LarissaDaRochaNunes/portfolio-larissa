Jogo de Adivinhação Numérica

Uma aplicação interativa desenvolvida em JavaScript puro (Vanilla JS) para praticar lógica de programação e manipulação do DOM. O sistema gera um número pseudoaleatório e gerencia as tentativas do usuário em tempo real, fornecendo feedbacks condicionais ("maior" ou "menor") para guiar o jogador até o acerto.

Verificador de idade

Aplicação web interativa que calcula a idade do usuário com base no ano de nascimento informado e identifica sua faixa etária e gênero. O sistema atualiza dinamicamente a interface, exibindo uma mensagem personalizada e uma imagem correspondente ao perfil gerado (bebê, jovem, adulto ou idoso). Ideal para demonstrar manipulação do DOM e lógica de validação de dados em JavaScript puro.
